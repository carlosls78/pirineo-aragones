# Guía de viaje · Pirineo Aragonés con Greta

Guía web autocontenida de un viaje de 16 días (1-16 julio 2026) por el Pirineo Aragonés con perro, organizada en tres campos base.

## Contenido

- **index.html** — Portada con enlaces a los tres campos base.
- **campo-base-1-valle-de-hecho.html** — Valle de Hecho (1-6 julio).
- **campo-base-2-broto.html** — Broto y Ordesa (6-11 julio).
- **campo-base-3-benasque.html** — Benasque y el Aneto (11-16 julio).

Mantén los cuatro archivos en la misma carpeta para que los enlaces internos funcionen. Abre `index.html` para empezar.
